# SnakeGame
### 국민대학교 강좌 'c++ 프로그래밍'의 과제
#### c++과 ncurces를 사용하여 스네이크 게임 제작
##### 2021.05. ~ 2021.06.19.
---
## 게임 설명
- 이미지

  
- 사과(빨간색)를 먹으면 뱀이 길어지고, 독(초록색)을 먹으면 뱀이 짧아집니다. 
- 벽(회색)에 부딪히거나 자신의 몸에 부딪히면 게임오버됩니다.
- 포탈(분홍색)을 타서 먼거리를 한번에 이동할 수도 있습니다.
- 목표한 만큼 뱀의 크기를 키우면 다음 스테이지로 넘어가게됩니다.
